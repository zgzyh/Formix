## 一、软件简介

Formix 是一个基于 FFmpeg 的桌面多媒体处理工具，当前包含以下主要功能：

1. 视频格式转换
2. 音频格式转换
3. 图片格式转换
4. M3U8 下载与转封装
5. 音视频分离
6. 音视频合成
7. 命令行页面（内置 FFmpeg / FFplay / FFprobe）
8. GPU 编码、背景设置、每日壁纸、在线更新

## 二、运行前准备

### 1. 安装 Python 依赖

如果你当前就在项目根目录，执行：

```bash
pip install -r requirements.txt
```

### 2. 准备 FFmpeg

本项目默认优先使用软件目录内的 FFmpeg，不依赖系统环境变量。

默认目录：

- `format_factory/FFmpeg/bin/ffmpeg.exe`
- `format_factory/FFmpeg/bin/ffprobe.exe`
- `format_factory/FFmpeg/bin/ffplay.exe`

说明：

- `ffmpeg` 是核心必须文件，没有它时转换功能不可用。
- `ffprobe` 用于文件信息识别、格式分析和部分页面的自动判断。
- `ffplay` 用于命令行页面直接播放媒体文件。
- 可以在软件“设置”页直接点击下载或更新 FFmpeg，程序会先下载到 `format_factory/ffmpeg_cache`，再自动解压到 `format_factory/FFmpeg`。

### 3. 启动方式

建议在项目根目录执行：

```bash
python -m format_factory.main
```

## 三、主界面说明

当前主界面包含以下分页：

1. 视频
2. 音频
3. 图片
4. M3U8
5. 音视频
6. 命令行
7. 设置

说明：

- “命令行”分页默认可在设置中关闭，关闭后菜单不显示。
- 大多数转换页的操作流程一致：添加文件、选择格式、选择输出目录、调整预设、开始转换。

通用特性：

- 支持拖拽文件到列表
- 支持批量处理
- 支持取消当前任务及后续排队任务
- 支持日志复制与清空
- 输出格式、预设、分辨率、语言、背景填充模式不支持鼠标滚轮误切换

## 四、视频转换

### 支持的输出格式

- `mp4`
- `m4a`
- `mkv`
- `avi`
- `mov`
- `webm`
- `flv`
- `gif`
- `m3u8`

### 功能说明

- 常见视频格式互转
- 可选分辨率缩放
- 可选预设与自定义参数
- 支持输出为 HLS 播放列表
- 支持 GPU 编码自动注入和不支持时自动回退

### 使用步骤

1. 添加视频文件或文件夹
2. 选择输出格式
3. 按需选择分辨率
4. 选择预设或自定义参数
5. 选择输出目录
6. 点击“开始转换”

### 说明

- `gif` 更适合短片段导出。
- 输出 `m3u8` 时，会自动创建 `原文件名_segments` 子目录保存播放列表和切片。
- 如果使用“自定义”参数，请确保你了解 FFmpeg 参数含义。

## 五、音频转换

### 支持的输出格式

- `mp3`
- `m4a`
- `aac`
- `wav`
- `flac`
- `ogg`
- `opus`

### 功能说明

- 常见音频格式互转
- 支持从带封面的媒体中提取音频
- 支持 `.ncm` 文件解密后继续转换
- 针对 `m4a`、`aac`、`wav` 做了容器与编码兼容性修正

### 使用步骤

1. 添加音频文件
2. 选择目标格式
3. 选择预设或自定义参数
4. 选择输出目录
5. 点击“开始转换”

### `.ncm` 说明

- 软件会先解密 `.ncm`，再按目标格式继续处理。
- 解密缓存会放在 `ncm_cache` 目录。
- 任务结束后程序会自动尝试清理缓存文件。

## 八、音视频页

音视频页包含两个子页：

1. 分离音视频
2. 合成音视频

### 1. 分离音视频

功能：

- 提取纯音频
- 提取纯视频
- 同时提取音频和视频

输出命名：

- 音频：`原文件名_audio.扩展名`
- 视频：`原文件名_video.扩展名`

### 2. 合成音视频

功能：

- 将一个音频文件批量合成到多个视频文件中

输出命名：

- `原视频名_merged.扩展名`

音频处理方式示例：

- 复制音频流
- 转为 M4A(AAC)
- 转为 AAC 192k
- 转为 MP3 192k
- 转为 FLAC
- 转为 Opus 96k

## 九、命令行页面

### 支持命令

- `ffmpeg`
- `ffplay`
- `ffprobe`

### 功能说明

- 可直接在软件内执行命令
- 支持带空格路径和带引号路径
- 支持命令历史记录
- 支持 `Ctrl + C` 中断运行
- 支持鼠标拖拽选择日志与报错文本
- 执行 `ffmpeg` 时底部显示进度条

### 使用方式

1. 打开“命令行”分页
2. 直接输入命令
3. 按回车执行
4. 用 `Ctrl + Enter` 换行
5. 用键盘上方向键回填历史命令

### 示例

```bash
ffmpeg -i "input.mp4" -c:v libx264 "output.mp4"
ffprobe "input.mp4"
ffplay "input.mp4"
```

说明：

- 命令行页面只允许执行 `ffmpeg`、`ffplay`、`ffprobe`，不会执行其他系统命令。
- 如果本地缺少 `ffprobe` 或 `ffplay`，页面会直接提示缺失。

## 十、设置页

设置页主要包括以下内容：

1. 外观
2. GPU 设置
3. 关于
4. 软件更新
5. FFmpeg 下载与更新

### 1. 外观

支持设置：

- 亮色 / 深色 / 自动跟随系统
- 语言切换
- 命令行页面启用与禁用
- 背景图片
- 背景图片透明度
- 毛玻璃模糊
- 背景填充模式
- 每日壁纸

背景填充模式包括：

- 无拉伸
- 拉伸填充
- 等比适应
- 等比填充（裁剪）

### 2. 语言切换

支持语言：

- 简体中文
- 繁體中文
- English
- 日本語
- 한국어
- 自动跟随系统

切换语言后会弹出提示框，并提供两个选项：

- `立即退出`
- `稍后自行退出`

选择“立即退出”后软件会自动关闭。

### 3. 每日壁纸

功能：

- 自动从在线接口获取壁纸
- 本地缓存当天壁纸
- 可手动刷新

说明：

- 手动清空背景图时，会同时关闭每日壁纸。
- 如果手动更换背景图，每日壁纸功能也会停止生效。

### 4. GPU 设置

支持：

- NVIDIA NVENC
- AMD AMF
- Intel Quick Sync
- 不使用 GPU

说明：

- 主要用于视频编码加速。
- 只有当前 FFmpeg 编译包含对应编码器时才会生效。
- 遇到不支持 GPU 的预设，程序会自动回退到 CPU。

### 5. 软件更新

功能：

- 手动检查新版本
- 查看更新公告
- 下载并准备安装更新

更新接口当前使用 GitHub Release Latest。

### 6. FFmpeg 下载与更新

功能：

- 下载或更新 FFmpeg
- 自动显示下载与解压进度
- 自动解压到 `format_factory/FFmpeg`
- 完成后删除压缩包

说明：

- Windows、Linux、macOS 会根据系统平台选择对应下载资源。
- 安装时会一并处理 `ffmpeg`、`ffprobe`、`ffplay`。

## 十二、常见问题

### 1. 软件启动后提示未找到 FFmpeg

请检查以下目录中是否存在对应文件：

- `format_factory/FFmpeg/bin/ffmpeg.exe`
- `format_factory/FFmpeg/bin/ffprobe.exe`
- `format_factory/FFmpeg/bin/ffplay.exe`

如果缺失，可到设置页下载。

### 2. 看不到媒体信息或格式识别结果

通常是 `ffprobe` 缺失导致。

### 3. 命令行页面无法执行 `ffplay`

通常是本地没有安装 `ffplay`，请重新通过设置页下载或更新 FFmpeg 套件。

### 4. GPU 已选择但没有加速

可能原因：

- 当前预设不支持 GPU
- FFmpeg 未编译对应 GPU 编码器
- 当前输出格式或编码方式不适合 GPU

### 5. 转换失败如何排查

建议按顺序检查：

1. 查看日志末尾错误信息
2. 检查输入文件是否损坏
3. 更换目标格式重试
4. 关闭 GPU 后重试
5. 清空自定义参数后重试
