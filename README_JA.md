## 1. 概要

Formix は FFmpeg をベースにしたデスクトップ向けマルチメディア処理ツールです。現在の主な機能は以下のとおりです。

1. 動画変換
2. 音声変換
3. 画像変換
4. M3U8 のダウンロードとリマックス
5. 音声と動画の分離
6. 音声と動画の合成
7. FFmpeg / FFplay / FFprobe 内蔵コマンドページ
8. GPU エンコード、背景設定、日替わり壁紙、オンライン更新

## 2. 使用前の準備

### 1. Python 依存関係のインストール

プロジェクトのルートで以下を実行します。

```bash
pip install -r requirements.txt
```

### 2. FFmpeg の準備

本アプリは既定でプロジェクト内の FFmpeg を優先して使用し、システム PATH には依存しません。

想定されるファイル:

- `format_factory/FFmpeg/bin/ffmpeg.exe`
- `format_factory/FFmpeg/bin/ffprobe.exe`
- `format_factory/FFmpeg/bin/ffplay.exe`

補足:

- `ffmpeg` は変換機能に必須です。
- `ffprobe` はメディア情報の解析と形式判定に使われます。
- `ffplay` はコマンドページでの再生に使われます。
- 設定ページから FFmpeg のダウンロードまたは更新も可能です。アーカイブは `format_factory/ffmpeg_cache` に保存され、その後 `format_factory/FFmpeg` に自動展開されます。

### 3. 起動方法

プロジェクトのルートで以下を実行します。

```bash
python -m format_factory.main
```

## 3. メイン画面

現在のメイン画面には次のページがあります。

1. 動画
2. 音声
3. 画像
4. M3U8
5. 音声/動画
6. コマンド
7. 設定

共通仕様:

- コマンドページは設定から無効化できます。
- 多くの変換ページは、ファイル追加、形式選択、出力先選択、プリセット調整、開始、という流れです。

共通機能:

- ファイルのドラッグ＆ドロップ
- バッチ処理
- 実行中タスクと待機中タスクのキャンセル
- ログのコピーとクリア
- 出力形式、プリセット、解像度、言語、背景フィルモードはマウスホイールで誤変更されません

## 4. 動画変換

### 対応出力形式

- `mp4`
- `m4a`
- `mkv`
- `avi`
- `mov`
- `webm`
- `flv`
- `gif`
- `m3u8`

### 機能

- 一般的な動画形式の相互変換
- 解像度の変更
- プリセットとカスタム FFmpeg 引数
- HLS プレイリスト出力
- GPU 引数の自動注入と非対応時の CPU フォールバック

### 手順

1. 動画ファイルまたはフォルダを追加
2. 出力形式を選択
3. 必要に応じて解像度を選択
4. プリセットまたはカスタム引数を選択
5. 出力先フォルダを選択
6. 開始をクリック

## 5. 音声変換

### 対応出力形式

- `mp3`
- `m4a`
- `aac`
- `wav`
- `flac`
- `ogg`
- `opus`

### 機能

- 一般的な音声形式の変換
- ジャケット付きメディアからの音声抽出
- `.ncm` の復号後変換
- `m4a`、`aac`、`wav` などのコンテナとコーデックの互換性補正

### `.ncm` に関する補足

- `.ncm` は先に復号され、その後ターゲット形式に変換されます。
- 一時ファイルは `ncm_cache` に作成されます。
- 完了後に自動削除が試行されます。

## 6. 画像変換

### 対応出力形式

- `jpg`
- `png`
- `webp`
- `bmp`
- `tiff`
- `ico`

### 機能

- 一般的な画像形式の相互変換
- 品質、圧縮、拡大縮小プリセット
- アイコンファイル出力

## 7. M3U8 ダウンロードとリマックス

### 対応入力

- オンライン `http://` / `https://` M3U8 URL
- ローカル `.m3u8` ファイル

### 対応出力形式

- `mp4`
- `mkv`
- `avi`
- `mov`
- `webm`

### 機能

- オンライン M3U8 のダウンロード
- ローカル M3U8 のリマックスまたは変換
- 一般的な動画コンテナへ直接出力

## 8. 音声/動画ページ

このページには 2 つのサブページがあります。

1. 分離
2. 合成

### 1. 分離

機能:

- 音声のみ抽出
- 動画のみ抽出
- 音声と動画を同時抽出

出力名:

- 音声: `original_name_audio.ext`
- 動画: `original_name_video.ext`

### 2. 合成

機能:

- 1 つの音声ファイルを複数の動画に一括合成

出力名:

- `original_video_name_merged.ext`

## 9. コマンドページ

### 対応コマンド

- `ffmpeg`
- `ffplay`
- `ffprobe`

### 機能

- アプリ内でコマンドを直接実行
- 引用付きパス、空白を含むパスに対応
- コマンド履歴
- `Ctrl + C` で現在のコマンドを停止
- マウスドラッグでログやエラーを選択可能
- `ffmpeg` 実行時は下部に進捗バーを表示

### 例

```bash
ffmpeg -i "input.mp4" -c:v libx264 "output.mp4"
ffprobe "input.mp4"
ffplay "input.mp4"
```

補足:

- このページでは `ffmpeg`、`ffplay`、`ffprobe` のみ実行可能です。
- `ffplay` または `ffprobe` が存在しない場合は、そのままエラー表示されます。

## 10. 設定

設定ページには以下の項目があります。

1. 外観
2. GPU 設定
3. 情報
4. ソフト更新
5. FFmpeg ダウンロード/更新

### 1. 外観

設定可能項目:

- ライト / ダーク / システム連動テーマ
- 言語切替
- コマンドページの有効化/無効化
- 背景画像
- 背景画像の透明度
- ブラー強度
- 背景フィルモード
- 日替わり壁紙

背景フィルモード:

- 引き伸ばしなし
- 引き伸ばし
- 比率維持で全体表示
- 比率維持で全体を埋める（トリミング）

### 2. 言語切替

対応言語:

- 簡体字中国語
- 繁体字中国語
- 英語
- 日本語
- 韓国語
- システムに従う

言語変更後は以下の 2 つの選択肢が表示されます。

- `今すぐ終了`
- `あとで自分で終了`

`今すぐ終了` を選ぶとアプリはそのまま終了します。

### 3. 日替わり壁紙

機能:

- オンライン API から壁紙取得
- 当日の壁紙をローカル保存
- 手動更新

### 4. GPU 設定

対応:

- NVIDIA NVENC
- AMD AMF
- Intel Quick Sync
- CPU のみ

補足:

- GPU 加速は主に動画エンコード用です。
- 現在の FFmpeg ビルドに必要なエンコーダが含まれている場合のみ有効です。
- 非対応プリセットは自動的に CPU にフォールバックします。

### 5. ソフト更新

機能:

- 新バージョン確認
- 更新内容表示
- 更新のダウンロードと準備

### 6. FFmpeg ダウンロード/更新

機能:

- FFmpeg のダウンロードまたは更新
- ダウンロードと展開の進捗表示
- `format_factory/FFmpeg` への自動展開
- 完了後に圧縮ファイルを自動削除

補足:

- Windows、Linux、macOS でそれぞれ適切な配布物を使用します。
- `ffmpeg`、`ffprobe`、`ffplay` はまとめて処理されます。

## 12. よくある問題

### 1. 起動時に FFmpeg が見つからない

以下のファイルが存在するか確認してください。

- `format_factory/FFmpeg/bin/ffmpeg.exe`
- `format_factory/FFmpeg/bin/ffprobe.exe`
- `format_factory/FFmpeg/bin/ffplay.exe`

存在しない場合は設定ページからダウンロードしてください。

### 2. メディア情報や形式判定が出ない

多くの場合 `ffprobe` が不足しています。

### 3. コマンドページで `ffplay` が実行できない

多くの場合 `ffplay` がローカルに存在しません。設定ページから FFmpeg セットを再取得してください。

### 4. GPU を選んでも加速されない

原因の例:

- 現在のプリセットが GPU 非対応
- FFmpeg に必要な GPU エンコーダが含まれていない
- 形式またはコーデック経路が GPU 向きではない

### 5. 変換に失敗する

次の順で確認してください。

1. ログ末尾のエラー行を確認
2. 入力ファイルが正常か確認
3. 別の出力形式を試す
4. GPU を無効にして再試行
5. カスタム引数を消して再試行
